## [1.7.0] — 2026-05-03

### Добавлено
- **Активные скиллы классов** — одна активная способность на ветку с кулдауном: `focus` (Новичок, 20с), `shield_bash` (Воин, 8с), `poison_stab` (Плут, 10с), `volley` (Лучник, 12с), `fireball` (Маг, 15с). Кнопка скилла в `#skill-zone`, кулдаун-бар обновляется каждые 100мс.
- **DoT эффекты** — яд (`poisonTicks`, `poisonDmg`) и горение (`burnTicks`, `burnDmg`) на мобах; обрабатываются в начале каждого тика `Combat.js`; `_applySkill()` вешает DoT при использовании скиллов. Стан (`stunTicks`) для `shield_bash`.

### Изменено
- **GameScene.js** рефакторинг: 764 строки → тонкий оркестратор (76 строк) + 3 mixin-файла: `scene/SceneFX.js` (227), `scene/SceneBackground.js` (175), `scene/SceneEntities.js` (264). Паттерн `installXxx(GameScene.prototype)`.
- **GameState.js** рефакторинг: методы save/load/autosave/hardReset вынесены в `core/GameStateSave.js` (89 строк) через mixin `installSave(GameState.prototype)`.
- `src/data/skills.js` — новый файл с таблицей `SKILLS_BY_BRANCH`.

## [1.6.0] — 2026-05-02

### Добавлено
- **Флаги мобов** — 4 механических архетипа: `shield` (щит 25% maxHp), `regen` (+2% HP/0.4с), `armored` (DEF×1.5, крит игнорирует броню), `swift` (speed×2, HP×0.5). Боссы гарантированно 1–2 флага; обычные мобы 30% шанс.
- **Элиты** — появляются на каждой 5-й волне (не-босс): HP×3, ATK×1.8, гарантированный редкий дроп, 1–2 флага. Имя `⚡ <Моб>`, жёлтый чип в BattleStrip.
- **Floating text polish** — MISS (серый) при dodge, БЛОК (синий) + урон на мобе при thorns, 🛡 ЩИТ СЛОМАН при разрушении щита, +heal при регене моба, screen shake при ударе босса.
- **Иконки мобов** — каждый из 20 типов (мобы + боссы) получил эмодзи-иконку в `MOB_ICONS`.
- **Иконки флагов на боевой сцене** — над мобом отображается `Гоблин 🛡⚡` (Phaser text).
- **Tooltip в BattleStrip** — при наведении на чип врага: HP с цветом, щит, атака, описание флагов.

### Исправлено
- Импорт сейвов версии v2 теперь работает (`SettingsMenu._importSave`: проверка `v !== 1` → `v !== 1 && v !== 2`).

### Изменено
- Нормальный удар в floating text: `#ffffff` → `#ffeecc` (теплее, лучше отличается от крита).
- `combat:waveStarted` теперь включает поле `isEliteWave`.

## [1.5.3] — 2026-04-27

### Баланс
- **Логарифмическое масштабирование мобов** — HP/ATK/DEF растут по `1 + k*ln(wave)` вместо `1.06^wave`. Плавная кривая, без экспоненциальной стены.
- **Нарастающий коэффициент** — k стартует с 0.55 и прибавляет 0.10 каждые 5 волн: wave 10 ≈ 2.50x, wave 40 ≈ 5.6x, wave 100 ≈ 12x.
- **XP и золото** сохраняют прежний экспоненциальный рост `1.06^wave` — награды не упали.
- **Штраф за смерть отменён** — −5% золота при гибели больше нет.

## [1.5.2] — 2026-04-25

### Добавлено
- **Milestone-уведомления** — при первом прохождении волны кратной 10 (10, 20, 30...) появляется флэш-оверлей с анимацией: «ВОЛНА X ПРОЙДЕНА!», отметка «🏆 НОВЫЙ РЕКОРД!» и золотой бонус `+wave×50🪙`.
- **Золотой бонус за рекорд** — за каждую новую рекордную milestone-волну начисляется `wave × 50` золота (с учётом goldMult игрока).
- **`state.maxWaveReached`** — новое поле в GameState, хранит максимально достигнутую волну за всё время (не сбрасывается при престиже); сохраняется в localStorage.

## [1.5.1] — 2026-04-24

### Исправлено / Оптимизация
- **Снижены подтормаживания** во время боя: `getStats()` теперь вычисляется один раз за тик вместо ~12 раз.
- **`addXp()` без level-up** больше не вызывает полный перерасчёт UI — вместо `statsChanged` бросается лёгкий `xpChanged`, обновляющий только XP-бар.
- **`BattleStrip`**: HP врагов обновляется точечно (меняем ширину fill-элемента), полный `innerHTML` пересчёт — только при смерти моба или новой волне.
- **`BattleStrip` + `GameScene`**: `maxHp` кешируется и обновляется только при реальном изменении статов, а не при каждом ударе.

## [1.5.0] — 2026-04-24

### Добавлено
- **Кросс-ветковые особые способности** — 11 классов depth 2–4 получили уникальные механики из других веток:
  - **Убийца** → Смертельный удар (1%) — профессиональный убийца знает куда бить
  - **Ниндзя** → Пробитие брони (4%) — удар в уязвимую точку сквозь любую броню
  - **Тень** → Вампиризм (4%) — питается жизнью поверженных врагов
  - **Призрак** → Маг. щит (5%) — нематериальная природа отражает магию
  - **Инквизитор** → Маг. щит (5%) — святая воля защищает от проклятий
  - **Тёмный Судья** → Смертельный удар (3%) — смертный приговор как привилегия
  - **Охотник на Ведьм** → Маг. щит (8%) — изучил магию, чтобы её уничтожать
  - **Гладиатор** → Шипы (7%) — шипастый щит наказывает атакующих
  - **Охотник за Головами** → Смертельный удар (2%) — знает единственный верный удар
  - **Пират** → Вампиризм (4%) — кровожадный морской разбойник
  - **Шаман** → Уворот (4%) — духи-хранители уводят удары

## [1.4.0] — 2026-04-21

### Добавлено
- **Пробитие брони (🏹 Pierce)** — уникальный стат ветки лучника. Снижает эффективную защиту моба на %. Накапливается по всей ветке, максимум у Снайпера → Орлиный Глаз → Меткий Стрелок (~35%). Cap 75%.
- **Смертельный удар (💀 Deathblow)** — шанс мгновенно убить моба одним выстрелом. Только ветка Снайпера (depth 2–4). Не работает на боссах. Cap 20%.

### Исправлено
- Сброс прогресса в настройках теперь работает корректно. Причина бага: `window.location.reload()` запускал `beforeunload`, который перезаписывал сейв обратно до перезагрузки страницы.

## [1.3.0] — 2026-04-21

### Баланс
- **Масштабирование мобов** смягчено: `1.08^(wave-1)` → `1.06^(wave-1)`.
- **Защита мобов** теперь растёт по корню волны (была линейной) — игрок не упирается в стену на поздних волнах.
- **Количество мобов**: максимум снижен с 10 до 8, прирост замедлен (1 каждые 4 волны вместо 3).
- **Боссы**: скорость атаки `×0.7` → `×1.1` (теперь чуть агрессивнее обычных мобов). Первые 4 босса: `atkMult 3` → `4`.

## [1.1.0] — 2026-04-21

### Добавлено
- **Уворот (Dodge)** — новый стат, шанс полностью уклониться от удара. Растёт через апгрейд «🌀 Уворот» (+2%/ур.), бонусы rogue-ветки (Плут→Ниндзя→Тень/Призрак до 25%), аксессуары. Cap 75%.
- **Вампиризм (Lifesteal)** — процент нанесённого урона восстанавливает HP. Апгрейд «🩸 Вампиризм» (+3%/ур.), bloodthirst-ветка (Кровожад→Вампир до 20%), оружие.
- **Шипы (Thorns)** — процент входящего урона отражается обратно мобу. Апгрейд «🌵 Шипы» (+3%/ур.), paladin-ветка (Паладин→Страж Веры до 31%), броня.
- Три новых апгрейда в магазине прокачки.
- Новые бонусы в дропе предметов: dodge на аксессуарах, lifesteal на оружии, thorns на броне.
- Новые строки в панели характеристик (скрыты пока стат = 0).

### Изменено
- **Формула очков престижа** пересмотрена: `floor(wave^1.45 / 15) + floor(level / 20)` вместо `floor(wave/2) + floor(level/2)`. Wave теперь доминирует нелинейно; фарм уровней на волне 1 не даёт значимых ПО.
